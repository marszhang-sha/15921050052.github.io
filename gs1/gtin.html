# GTIN

# 什么是GTIN

作为GS1系统的主要组成部分之一，全球贸易项目编号®（GTIN®，Global Trade Item Number®）是全球唯一的GS1标识符，用于标识贸易项目。GTIN由产品的品牌持有者分配，用于在产品在全球供应链中流通时识别产品。

什么是Trade Item 在供应链的任一点上，任何一个可被定价、订购或者开票的服务或者商品叫做贸易项目(Trade Item)。贸易项目包括独立产品以及所有以其他包装 配置形式出售的产品（例如，双包;箱式;托盘;等）。每个包装级别由唯一的GTIN码识别。例如，品牌 X 将使用不同的 GTIN 来唯一标识 他们单包纸巾，四包装纸巾，和以及将要在供应链中流通的其他包装的纸巾，

# GTIN应用范围

GTIN 可用于需要识别产品或服务的任何位置， 包括互联网、商业交易、IT 系统和物理产品本身。GS1 标准定义了如何格式化 并在各种应用中建立 GTIN，以便使用相同的 GTIN 来识别产品并满足不同的需要：

  • 物理产品：GTIN 可编码为条形码和射频识别 （RFID）标签并贴在产品上，这样商品在供应链中流通时能够准确识别。

  • 电子商务：GTIN 可用于电子商务交易（例如，采购订单、提前发货通知、发票等）支持订单到现金的业务流程。GTINs 是大多数电子数据（EDI）交换业务的必须的一个数据元素，以减少错误并支持机器到机器的处理。

  • 互联网应用：GTIN 可以作为统一资源标识符 （URI） 显示，用以支持基于 Internet 的应用程序和数据共享。GS1 电子产品代码 （Electronic Product Code ，EPC®）标准包括 GTIN 的两种 URI 数据格式。这些 EPC UPN 提供标准化数据格式，与互联网的标准化数据格式一致，以便不仅支持基于 Internet 的数据共享还支持"虚拟"产品（例如：可被下载的下书籍和音乐）和没有条形码阅读器，没有EPC扫描仪的"虚拟"销售点

    • LGTIN EPC (a Uniform Resource Name (URN) for GTIN plus batch/lot number)
    • SGTIN EPC ((a Uniform Resource Name (URN) for GTIN plus serial number)

  • 数据库和 IT 系统：GTIN 提供可用于所有系统的产品标识符（例如采购、库存管理、物流、分析、报告等）。这将保持系统之间的连接，以便它们可以共同提升数据的质量和数量来支持运营流程，当然也包括商业智能和分析数据。

# GTIN 结构

GTIN 的长度可以是 8、12、13 或 14 位数字，以便适应不同的应用和产品约束。这些 GTIN 结构分别称为 GTIN-8、GTIN-12、GTIN-13 和 GTIN-14。 每个 GTIN 都是由不同段组成的数字字符串。GTIN 段包括：

  • 指示符位（Indicator Digit）：指示符位表示包装级别，用来识别同一个项目参考号的不同包装级别。该字段由取值范围为1-9，仅在 GTIN-14 中使用。
  • 公司前缀（GS1 Company Prefix）:GS1 组织向公司颁发的全球唯一代码，用来生成GS1识别符（例如：GTIN代码）。根据公司需求不同，分配到的公司前缀长度不同（例如中国690-699，不同段的公司前缀长度不同）。
  • 项目参考号（Item Reference）：商品厂商为标识交易项目而分配的编号。项目参考号长度因 GS1 公司前缀长度的不同而变化。
  • 校验位（Check Digit）:根据GTIN码之前的位数通过计算得出的一位数字，用以保证数据的完整性。

# GTIN-8

The GTIN-8 is the only GTIN that can be used in EAN-8 barcodes. Components include:

  • Seven digits containing a GS1-8 Prefix and the Item Reference
  • One digit representing the Check Digit

# GTIN-12

The GTIN-12 is the only GTIN that can be used in UPC-A barcodes. Components include:

  • Eleven digits containing your GS1 US-Issued U.P.C. Company Prefix and the Item Reference, which is assigned by your company
  • One digit representing the Check Digit

# GTIN-13

The GTIN-13 is the only GTIN that can be used in EAN-13 barcodes. Components include:

  • Twelve digits containing your GS1 US-Issued GS1 Company Prefix and the Item Reference, which is assigned by your company
  • One digit representing the Check Digit

# GTIN-14

The GTIN-14 is used in ITF-14, GS1-128 (formerly UCC/EAN-128), GS1 DataBar®, and DataMatrix symbols as well as EPCs. Components include:

  • One digit representing the Indicator Digit to indicate packaging level
  • Twelve digits containing your GS1 US-Issued GS1 Company Prefix and the Item Reference which is assigned by your company
  • One digit representing the Check Digit

# GTIN关键特征

  • 全球性：GTINs 是一个全球标准，可由世界上的任何公司分配，可以在世界任何地方使用。 • 多行业：GTIN 可用于所有业务部门，支持任何产品（例如医疗保健产品、百货产品、零售产品等）使用相同的标准来识别标识符。 • 独特性：用于分配 GTIN 的标准化结构和规则有助于确保每个 GTIN全球唯一。 • 包装层次结构：将指标数字包含在 GTIN-14 结构中，使 GTIN 能够用于在所有包装级别上唯一识别贸易项目。 • 数据完整性：包含校验数字支持 GTIN 的唯一性和完整性结构

# GTIN与商业智能应用

GS1 标准提供 GTIN 的格式和结构，从而在可以各种需要识别产品和信息的平台上使用：数据库和系统、物理产品标记、电子商务交易和互联网应用。这使得贸易伙伴能够使用相同的标识符，跨平台识别产品 - 支持数据驱动的组织利用数据来优化商业智能和改进业务流程。

# GTIN 如何分配给商品

制造商或品牌所有者负责在为其产品分配 GTIN作为分配。 品牌所有者是拥有贸易项目规格的组织，无论在哪里或由谁制造。 GS1 系统提供清晰、结构化的数据标准和管理规则，制造商在分配GTIN时遵循这个规则，以确保其GTIN 具有全球唯一性并且格式一致。

分配 GTIN 有两个基本步骤：

  • GS1 组织向制造商颁发 GS1 公司前缀。GS1 公司前缀提供生成 GS1 识别符的基础。
  • 制造商根据其 GS1 公司前缀、GS1 标准以及GTIN 管理规则分配或者生成GTIN码。当制造商分配 GTIN 时，他们还会定义一组与GTIN有关的产品数据。这些产品属性描述定义了所有产品实例统一的产品的核心数据（例如尺寸、颜色、品牌信息等）。GTIN 及其关联 的属性保存在数据库中，并在供应链合作伙伴之间共享，以支持供应链操作和应用系统。

# GTIN 优势

作为 GS1 系统的关键组件，GTIN 已经是全球贸易项目的标准。 GTIN使贸易伙伴能够使用唯一的全球产品识别标准高效管理产品信息。 这有利于贸易伙伴之间的准确的沟通产品信息，以支持供应链运营和优化商业智能。 简单地说，使用 GTIN 可简化供应链管理并促进准确性、速度和效率。

  • 推动电子商务：在电子商务领域使用 GTIN， 促进贸易项目和相关信息的全球流动。

  • 增强兼容性：因为GTIN的跨业务领域特性，公司贸易之间识别产品和服务可以兼容

  • 提高准确性：使用 GTIN 可在结账、仓库或医院，改善扫描方式。同时对于准确的库存控制和订单补货也至关重要。

  • GDSN：通过GTIN 进行产品识别，用户能够利用 GS1 全球数据同步网络™ （GDSN®））来管理产品信息。GDSN 提供持续、自动化的数据管理方法，有助于确保产品信息在供应链合作伙伴之间相同，提高了数据准确性，并降低了供应链成本。

  • 简化供应链管理：通过准确识别产品以及信息流，GTIN加强了供应链合作伙伴间的业务沟通

  • 提高信息质量：使用GTIN作为产品信息的链接，通过确保供应链合作伙伴之间的产品信息一致性，提高了信息质量。 这对内部和外部业务流程大有裨益。

# 计算校验位

# STEP ONE:

Suppose you want to find the Check Digit for the GTIN-12 (U.P.C.) Number 61414121022. Set up a table with 12 columns, and put the number 61414121022 into Positions One through Eleven. Position Twelve will be blank because it is reserved for the Check Digit.

# STEP TWO:

Add the numbers in Positions One, Three, Five, Seven, Nine, and Eleven: (6+4+4+2+0+2=18)

# STEP THREE:

Multiply the result of Step Two by three: (18x3=54)

# STEP FOUR:

Add the numbers in Positions Two, Four, Six, Eight, and Ten: (1+1+1+1+2=6)

# STEP FIVE:

Add the results of Step Three and Step Four: (54+6=60)

# STEP SIX:

The Check Digit is the smallest number needed to round the result of Step Five up to the nearest multiple of 10. In this example, the Check Digit is 0.

# 编码类型

# EAN/UPC条码

EAN/UPC条码包括EAN-13、EAN-8、UPC-A和UPC-E，如图3-6所示。通过零售渠道销售的贸易项目必须使用EAN/UPC条码进行标识。同时这些条码符号也可用于标识非零售的贸易项目。

  • EAN条码: EAN条码是长度固定的连续型条码，其字符集是数字0~9。EAN条码有两种类型，即EAN-13条码和EAN-8条码。
  • UPC条码 UPC条码是一种长度固定的连续型条码，其字符集为数字0~9。UPC码起源于美国，有UPC-A条码和UPC-E条码两种类型。 根据国际物品编码协会（GS1）与原美国统一代码委员会（UCC）达成的协议，自2005年1月1日起，北美地区也统一采用GTIN-13作为零售商品的标识代码。但由于部分零售商使用的数据文件仍不能与GTIN-13兼容，所以产品销往美国和加拿大市场的厂商可根据客户需要，向编码中心申请UPC条码。

# ITF-14条码

ITF-14条码（示例见图）只用于标识非零售的商品。ITF-14条码对印刷精度要求不高，比较适合直接印制（热转印或喷墨）在表面不够光滑、受力后尺寸易变形的包装材料上。因为这种条码符号较适合直接印在瓦楞纸包装箱上，所以也称“箱码”。关于ITF-14条码的说明，请查阅GB/T 16830-2008 《商品条码 储运包装商品编码与条码表示》国家标准。

# UCC/EAN-128条码

UCC/EAN-128条码（如图3-8）由起始符号、数据字符、校验符、终止符、左、右侧空白区及供人识读的字符组成，用以表示GS1系统应用标识符字符串。UCC/EAN-128条码可表示变长的数据，条码符号的长度依字符的数量、类型和放大系统的不同而变化，并且能将若干信息编码在一个条码符号中。该条码符号可编码的最大数据字数为48个，包括空白区在内的物理长度不能超过165㎜。UCC/EAN-128条码不用于POS零售结算，用于标识物流单元。

# EPC

EPC标签是射频识别技术中应用于GS1系统EPC编码的电子标签，是按照GS1系统的EPC规则进行编码，并遵循EPCglobal制定的EPC标签与读写器的无接触空中通信规则设计的标签。EPC标签是产品电子代码的载体，当EPC标签贴在物品上或内嵌在物品中时，该物品与EPC标签中的编号则是一一对应的。
