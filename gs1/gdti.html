# 什么是GDTI

全球文件类别标识码（Global Document Type Identifier，GDTI）是GS1标识的组成之一，用于识别不同类型的文件。

该唯一标识符由GS1公司前缀，文档类型，校验位和可选的序列号组成。

注意：“文档”一词广泛用于任何纸质或数字文件。

# GDTI的用途

GDTI是用于识别文档的简单工具。 如有需要，GDTI可以唯一地标识文档。 文档的发行者通常对文档中包含的所有信息负责，包括条形码和人类可解读（Human Readable Interpretation，HRI）信息。

当认为维护文件的记录具备重要性的时候，公司可以给文件分配GDTI。 GDTI将提供指向文档“主”副本的存储库的链接。 如果您的企业发布需要跟踪和控制的文档，则可以使用GDTI启用此功能。 GDTI可以由公司在世界任何地方分配，并且可以在世界任何地方使用。 任何需要跟踪的重要文档都可以使用GDTI作为关键字识别，并且应用程序也可以使用GDTI来获得相关的信息。

可以使用GDTI识别的文件类型的举例如下：

  • 识别土地产权注册文件
  • 纳税申报
  • 电子消息
  • 保单
  • 考卷
  • 发货单
  • 护照
  • 数字文件
  • 驾驶执照
  • 电视广告的标识
  • 内部发票

GDTI可以

  • 使用GS1-128或者GS1 DataMatrix生成
  • 使用具备RFID功能的EPC标签进行编码
  • 打印在文档上，作为一种识别方法或用于详细信息检索。

# 使用GDTI的业务收益

使用GDTI识别重要文档有许多益处：

  • GDTI的结构及其分配规则由GS1®管理，GS1®是一个非营利性标准化组织，该组织可以提供实施指南，业务示例和支持以及标准的维护.
  • GDTI是一个简单的编码系统，几乎具备无限的容量，帮助文档编号和识别。
  • GDTI已经是用于识别重要文档的既定标准，它使用与许多其他业务应用程序相同的GS1标准。如果您已经在使用GS1标准来标识产品或位置，则很容易扩展使用范围来标识重要文档。
  • GDTI在全球范围内是独一无二的。GDTI可以在全球范围内使用，而无需贸易伙伴分配专有编号以确保唯一性。
  • 如果需要，GDTI可以唯一地标识文档。
  • GDTI使公司可以扫描文档以检索或跟踪关键数据。
  • GDTI允许发行人而不是第三方分配GDTI。
  • GDTI防止错误识别和欺诈。
  • GDTI是一个易于定义的数据结构，具有完整性检查功能，可以简化数据的处理和传输。
  • GDTI可以用GS1-128或GS1 DataMatrix条形码符号和/或支持EPC的RFID标签以及人类可读的文本进行编码

# GDTI结构

GDTI由以下部分组成：

  • 应用程序标识符（AI）：表示GDTI的AI（253）
  • GS1公司前缀：分配给GS1成员公司的全球唯一编号
  • 文档类型：分配用于标识以下类型的编号：文档
  • 校验位：对10取模的计算结果
  • 序列号：用于单个文档的可选字母数字标识
AI GS1 Company Prefix Document Type Check Digit Serial Component (Optional)
(253) N1 N2 N3 N4 N5 N6 N7 N8 N9 N10 N11 N12 N13 X1 variable X17

注意：GDTI是一个13位数字，包括校验位，但不包括可选的序列号。GS1公司前缀和文档类型的总长度将始终为12位数字。

# 使用 GDTI的步骤:

1.从GS1公司前缀开始。GS1公司前缀是GS1成员组织向您的公司发行的全球唯一编号

2.创建文件类型。“文档类型”是由您（GS1公司前缀的持有者）分配的编号，用于唯一标识公司中的文档类型。文档类型的长度随GS1公司前缀长度的变化而变化。请注意，GS1公司前缀和文档类型的总长度始终为12位数字。

注意：按顺序分配文档类型编号，并且不要在数字中加入任何逻辑。因为当前的一组意义可能会在未来失效，比如公司重组，被其他公司收购或者合并，或者其他员工负责了GDTI工作。

3.计算校验位。校验位是计算得出的一位数，用于确保数据完整性。

4.（可选）分配序列号。列号使您跟踪每种文档类型的特定实例（例如，不仅是土地产权类型文件，而且具体的土地产权注册文件）。您可以为每个GDTI分配最多17个字符的字母数字序列号。同样，我们建议顺序分配序列号，序列号本身不要有任何业务逻辑。
